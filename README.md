🦸 Marvel App

A React-based web application for exploring Marvel characters, comics, and related content using the Marvel API.

👉 Live Demo: https://itssonyamisyura.github.io/marvel_app/

👉 Repository: https://github.com/itssonyamisyura/marvel_app

🚀 About

This project is a frontend application built with React that allows users to browse and search Marvel characters and view detailed information about them.

The app demonstrates working with APIs, routing, and dynamic UI rendering.

Applications like this typically use the official Marvel API to fetch real-time data about characters, comics, and series

✨ Features
🔍 Search Marvel characters
🧑 View detailed character information
📚 Explore comics and related content
⚡ Fast and responsive UI
📱 Mobile-friendly design

🛠️ Technologies Used
React
JavaScript (ES6+)
CSS / SCSS
REST API (Marvel API)
React Router
