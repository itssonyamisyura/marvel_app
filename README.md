# 🦸 Marvel App

A React-based web application for exploring Marvel characters, comics, and related content using the Marvel API.

👉 **Live Demo:** https://itssonyamisyura.github.io/marvel_app/  
👉 **Repository:** https://github.com/itssonyamisyura/marvel_app  

---

## 🚀 About

Marvel App is a frontend project built with React that allows users to search Marvel characters, view detailed character information, and explore related comics.

The app demonstrates:

- Working with APIs  
- Routing and navigation  
- Dynamic UI rendering  


The project focuses on working with external APIs, dynamic routing, reusable components, and responsive UI.

---

## ✨ Features

- 🔍 Search Marvel characters  
- 🧑 View detailed character information  
- 📚 Explore comics and related content  
- ⚡ Fast and responsive UI  

---

## 🛠️ Technologies Used

- React  
- JavaScript (ES6+)  
- CSS / SCSS  
- REST API (Marvel API)  
- React Router
